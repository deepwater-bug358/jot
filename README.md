## 📝 Using markdown

jot uses markdown, which means you can format text with plain symbols.

Common markdown items:

- `#` for a title
- `##` for a section heading
- `-` for a bullet list
- `**bold text**` for bold text
- `*italic text*` for italic text
- `[link text](https://example.com)` for links

This keeps writing fast and simple.

## 💬 Inline comment threads

Inline comment threads let you leave notes next to the text they relate to.

Use them for:

- feedback on a paragraph
- questions about a section
- review notes on a draft
- follow-up items for a team member

A good comment thread stays short and clear. Keep each thread tied to one point so it is easy to read later.

- right-click the file

- apply the change if it appears
- try opening the app again

If the app opens but looks blank:

- refresh the page if it runs in a browser
- reopen the app
- check that the folder or file path is correct

If your notes do not appear:

- confirm you opened the right folder
